# Technical Radar

**Built for a Google Hackathon using Antigravity + Google Gemini API**

Technical Radar is an AI-powered dashboard that helps developers, founders, and students discover which technologies are rising, relevant, and worth learning.

The tech world changes fast—new frameworks, AI tools, startups, and developer platforms launch constantly. Most people struggle to know what matters and what is just hype. Technical Radar solves this by transforming raw internet updates into clear insights.

## How It Works

Technical Radar uses the **Google Gemini API** as its intelligence engine.

Gemini helps:

- Analyze latest tech news and product launches  
- Detect repeated trends across sources  
- Filter noise and low-value content  
- Categorize tools into AI, Frontend, Backend, DevOps, Cloud, etc.  
- Summarize why each trend matters  
- Rank technologies by momentum and relevance  

Instead of reading dozens of blogs or communities manually, users get a clean radar of important movements in tech.

## Why Antigravity

Antigravity was used to rapidly build and ship the product during hackathon speed. It enabled faster prototyping, UI creation, and iteration so more focus could go into product logic and Gemini integration.

## Main Features

- Interactive radar dashboard  
- Trend discovery by category  
- AI-generated summaries  
- Momentum-based ranking  
- Searchable technology insights
